# jamfpro_macos_onboarding_settings (Resource)


## Example Usage
```terraform

  enabled = true

  onboarding_items {
    entity_id                = "1"
    self_service_entity_type = "OS_X_POLICY"
    priority                 = 1
  }

  onboarding_items {
    entity_id                = "5"

  }

}
```
